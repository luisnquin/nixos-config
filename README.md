# NixOS environment configuration

Installation:
```
$ git clone https://github.com/luisnquin/nixos-config.git ~/.dotfiles/
$ sudo ln -s ~/.dotfiles/ /etc/nixos/
$ sudo nixos-rebuild switch
```

Manage updates with:
```
$ sh update.sh
```
